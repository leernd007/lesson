# Хід роботи

## 1. Відкрити редактор

1. Відкрити браузер.
2. Перейти за посиланням: https://codepen.io/pen
3. На сторінці відкриється редактор з кількома полями:
   - HTML
   - CSS
   - JavaScript

Для цієї роботи використовуємо тільки **HTML**.

---

# Завдання 1. Створення простої вебсторінки

У поле **HTML** вставити код:

```html
<h1>Мій перший вебсайт</h1>

<p>Мене звати Ім'я.</p>

<p>Я навчаюсь у 8 класі.</p>

<h2>Моє хобі</h2>

<p>Я люблю слухати музику, дивитися фільми та працювати за комп'ютером.</p>
```

Після вставлення коду внизу з’явиться результат сторінки.

Змініть текст:

- впишіть своє ім’я  
- змініть опис хобі  

---

# Завдання 2. Додавання зображення

Додайте на сторінку зображення.

Приклад коду:

```html
<img src="https://upload.wikimedia.org/wikipedia/commons/3/3c/Shaki_waterfall.jpg" width="300">
```

Спробуйте:

- змінити розмір зображення
- знайти інше зображення в Інтернеті

---

## Завдання 3. Створення списку

Додайте список своїх улюблених занять.

```html
<h2>Мої улюблені заняття</h2>
<ul>
<li>Грати у футбол</li>
<li>Слухати музику</li>
<li>Дивитися фільми</li>
<li>Грати у комп'ютерні ігри</li>
</ul>
```

Спробуйте відредагувати і додати ще кілька пунктів списку.

## Завдання 4. Додавання кольору сторінки

У поле CSS додайте код:

```css
body {
  background-color: lightblue;
}
```

Спробуйте змінити колір сторінки на:

1. yellow
2. lightgreen
3. lightgray

## Завдання 5. Створення другого заголовка

Додайте новий розділ на сторінці.
      
```html
<h2>Моя школа</h2>
<p>Я навчаюся у школі № ...</p>
<p>Мій улюблений предмет — інформатика.</p>
```

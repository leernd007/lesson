# Завдання: Створення зимового відео в Canva

---

## Кроки виконання

1. **Створення дизайну**
   - Відкрий [Canva](https://www.canva.com/)
   - Натисни **Create a design → Video**
   - Обери формат **1920×1080 (Horizontal / Горизонтальне)**

2. **Додавання фону**
   - Вкладка **Photos / Фото**
   - Знайди та додай картинку з зимовою природою
     - Приклад пошуку: `winter forest`, `snow landscape`
   - Розтягни картинку на весь екран

3. **Додавання падаючого снігу**
   - Завантаж анімований сніг за цим [посиланням](https://github.com/leernd007/lesson/blob/main/snow.gif)
   - Перетягни його поверх фону
   - При необхідності налаштуй **Transparency / Прозорість**, щоб сніг виглядав природніше

4. **Додавання тексту**
   - Вкладка **Text / Текст** → додай заголовок
   - Введи текст: `Winter is coming`
   - Виділи текст → вкладка **Animate / Анімація**
     - Обери ефект появи (наприклад, `Fade` або `Rise`)
   - Встанови **Timing / Час появи** на **2 секунди**  

5. **Перегляд та збереження**
   - Переглянь відео, перевір чи текст з’являється в потрібний момент
   - Натисни **Share → Download**
   - Обери формат **MP4 Video**

# Приклад 1: Відео-анонс події

## Мета
Створити відео-анонс події (концерт, вебінар), додати фон, анімовані елементи і текст з деталями.

## Ключові моменти
- **Фон:** відповідний тематиці (сцена, офіс, музика)  
- **Елемент:** анімований банер або декорації  
- **Текст:** дата, час, назва події з анімацією

# Приклад 2: Осінній відеопост з падаючим листям

## Мета
Створити коротке горизонтальне відео з осіннім фоном, анімованим падаючим листям і текстом, що з’являється через 3 секунди.

## Ключові моменти
- **Фон:** осінній парк або ліс  
- **Елемент:** завантажене відео падаючого листя  
- **Текст:** “Autumn vibes” з ефектом появи через 3 секунди

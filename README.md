## Завдання

### 1. Підбір зображень
- Створи папку **GIF_Animation**.
- Знайди або використай існуючі:
  - **1 фотографію** (фон);
  - **1 кліпарт** (об’єкт для руху).

---

### 2. Обробка зображень
Відкрий кліпарт у веб-версії Photoshop:  
https://www.photopea.com/

**Виконай редагування кліпарту:**
- зміни розмір;
- зроби фон прозорим (інструмент *Magic Wand / магічне виділення*);
- збережи файл у форматі **PNG**.

**Обробка фотографії:**
- зменш розмір;
- за потреби відкоригуй яскравість;
- збережи файл у форматі **JPG**.

---

### 3. Підготовка фреймів для анімації
- Створи **6–8 копій** кліпарту.
- У кожному файлі змінюй положення об’єкта  
  (наприклад, рух зліва направо).
- Збережи файли з назвами:
  - `frame1.png`
  - `frame2.png`
  - `frame3.png`
  - `frame4.png`  
  - …

---

### 4. Створення GIF-анімації
- Використай онлайн-сервіс:  
  https://ezgif.com/maker
- Завантаж усі фрейми у правильному порядку.
- Збережи анімацію у форматі **GIF** з назвою  
  **animation.gif**.
